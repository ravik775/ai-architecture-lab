# Technology Architecture

## Runtime
- Java 17
- Spring Boot
- Spring Cloud Gateway
- Eureka
- RabbitMQ
- Redis
- Docker Compose
- Kubernetes

## Why RabbitMQ
RabbitMQ is suitable for enterprise workflow messaging, reliable queues, routing, retries, and dead-letter queues.

## Why Redis
Redis provides fast status reads, reducing database load and improving customer-facing latency.

## Why Eureka
Eureka demonstrates service discovery and dynamic routing for microservices in a laptop-friendly way.
